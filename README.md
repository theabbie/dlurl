# dlurl  
  
Download URL and return File Path

![dlurl logo](https://user-images.githubusercontent.com/17960677/98069898-fbdd4c80-1e85-11eb-982e-5c9fb856db4f.png)

## How to use

```js
var dl = require("dlurl");

(async function() {
var path = await dl(url[,name]);
console.log(path);
})();
```
